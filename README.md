# Cactus Financias (MVP)

## Execução rápida
```bash
make install
```

Após qualquer alteração de código:
```bash
make deploy
```

## Comandos principais
- `make help`
- `make deploy`
- `make down`
- `make logs`
- `make lint`
- `make in SERVICE=backend`
