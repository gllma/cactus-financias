Tudo em português (pt-BR), exceto código (classes, variáveis, métodos) que é em inglês.

## Conventional Commits

Formato obrigatório:
```
<tipo>(<escopo>): <descrição em português>

[corpo opcional]

[rodapé opcional]
```

### Tipos

| Tipo | Quando usar | Exemplo |
|---|---|---|
| `feat` | Nova funcionalidade | `feat(admin): implementar endpoint de aprovação de cadastro` |
| `fix` | Correção de bug | `fix(trip): corrigir cálculo de prioridade composta` |
| `docs` | Documentação | `docs(api): atualizar anotações Scramble nos controllers` |
| `style` | Formatação (Pint) | `style: formatar código com Laravel Pint` |
| `refactor` | Refatoração sem mudança funcional | `refactor(auth): extrair validação de token para middleware` |
| `perf` | Melhoria de performance | `perf(route): otimizar query de rotas com eager loading` |
| `test` | Testes | `test(trip-booking): adicionar testes de transição de estado` |
| `build` | Build/dependências | `build: adicionar dedoc/scramble via composer` |
| `ci` | CI/CD | `ci: adicionar etapa de pest no pipeline` |
| `chore` | Tarefas diversas | `chore: atualizar .gitignore` |

### Escopo

Usar o nome do Package ou módulo: `auth`, `trip`, `route`, `admin`, `user`, `person`, `address`, `bus`, `favorite`.

### Exemplos completos

```bash
feat(admin): implementar endpoints de gestão de cadastros PNE

- Criar RegistrationController com list, detail, approve, reject, request-correction
- Criar services de transição de estado com validação
- Adicionar permissões admin.registration.*
- Adicionar testes de transição válida e inválida

Ref: VM-014
```

```bash
fix(trip-booking): bloquear cancelamento com menos de 12h de antecedência

O prazo agora é configurável via config('vivamobil.cancellation_deadline_hours').
Anteriormente estava hardcoded como 12.
```

## Regras

- NUNCA incluir "Co-Authored-By", "Co-authored-by", "Generated by" ou qualquer tag de autoria/geração
- NUNCA fazer push sem revisão — commitar local, revisar, depois push
- Commits atômicos: uma feature/fix por commit
- Referenciar história Jira no corpo quando aplicável (ex: `Ref: VM-014`)

## Branches

Naming em português com prefixo:
- `feature/endpoints-gestor-cadastros`
- `fix/calculo-prioridade-composta`
- `refactor/extrair-validacao-token`
- `test/testes-transicao-estado-trip`

## Qualidade antes de commit

```bash
make lint          # Formatar com Pint (PSR-12)
make pest          # Rodar testes
```

## Testes (describe/it)

Nomes em português:
```php
describe('Aprovar cadastro PNE', function () {
    it('aprova cadastro com status pendente', function () { ... });
    it('rejeita aprovação quando status é reprovado', function () { ... });
});
```
